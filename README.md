# Structify3D

## Descrição

Structify3D é uma aplicação web que permite a criação de projetos de arquitetura 3D.

## Tecnologias Utilizadas

- Next.js
- Tailwind CSS
- Lucide
- React
- Node.js
- Cloudflare D1
- Cloudflare R2
- Cloudflare Workers

## Como executar a aplicação

1. Clone o repositório
2. Instale as dependências
3. Execute a aplicação
